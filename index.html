Setup guide
Mac
  1. Download the Claude desktop app and sign in.
  2. Upgrade to Claude Pro inside the app.
  3. Open Terminal and run:xcode-select --installOr get Git from git-scm.com
  4. Fully quit Claude via the menu bar or dock, then reopen.
  5. Optional: Settings → Claude Code → Allow bypass permissions.
Windows
  1. Download the Claude desktop app and sign in.
  2. Upgrade to Claude Pro inside the app.
  3. Download Git for Windows from git-scm.com — default settings are fine.
  4. Fully quit Claude via the system tray, then reopen.
  5. Optional: Settings → Claude Code → Allow bypass permissions.
